# 🌌 Astronomical Data Analysis using Python & Computer Vision

This project was developed as part of a hackathon where I secured **1st place**.

The hackathon involved solving multiple problems related to astronomical data using Python, Computer Vision, and Data Analysis.

---

## 🚀 My Contributions

I independently worked on the following problems:

### 1. Cosmic Ray Detection
- Detected noise artifacts in space images using image processing
- Applied median filtering, thresholding, and contour detection

### 2. Moon Crater Detection
- Detected lunar craters using Hough Circle Transform
- Identified circular structures from surface images

### 3. Galaxy Detection (Deep Field)
- Detected galaxies from deep field astronomical images
- Used adaptive thresholding and contour filtering
- Identified 400+ galaxies

### 4. Star Detection & Cluster Density Analysis
- Detected stars using connected components
- Calculated nearest neighbor distance
- Classified clusters as dense/sparse

### 5. Exoplanet Detection (Light Curve Analysis)
- Analyzed time-series data using Lightkurve
- Detected periodic dips indicating potential exoplanets
